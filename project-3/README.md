# ShopEase EV — Interactive E-Commerce Website

Frontend Development Project 3 – Interactive Web Elements.

## Files
- `index.html` — page structure (Header, Hero, Categories, Products, About, Contact, Footer)
- `style.css` — styling, hover effects, responsive layout, dark mode theme
- `script.js` — all interactivity (DOM Manipulation + Event Listeners)

## Interactive Features
- **Dark Mode Toggle** — `document.body.classList.toggle("dark-mode")`
- **Shopping Cart Counter** — Buy Now buttons update a live cart count in the header
- **Product Search** — live filtering of product cards as you type
- **Product Details Toggle** — View Details / Hide Details on each product card
- **Contact Form** — submits without reloading the page and shows a confirmation message

## How to Run
Just open `index.html` in any browser — no build step or server required.
