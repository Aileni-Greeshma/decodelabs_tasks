# ShopEase EV – Electric Scooty & Spare Parts Store 🛵⚡

> **Frontend Development Project 1 – Static Webpage Design**

A modern, fully static e-commerce landing page for electric scootys and EV spare parts, built using only **HTML5** and **CSS3**.

---

## 🖥️ Live Preview

Open `index.html` directly in any browser to view the project.

## 🚀 Features

- ✅ Built with **HTML5** and **CSS3** only — no JavaScript
- ✅ Semantic HTML tags: `header`, `nav`, `main`, `section`, `article`, `footer`
- ✅ Modern glassmorphic header with sticky navigation
- ✅ Full-screen hero section with electric scooty banner
- ✅ **6 Product Cards** — 3 Electric Scootys + 3 Spare Parts
- ✅ **6 Category Cards** — High-Speed Scootys, Eco Commuters, Batteries, Consoles, Tyres, Motors
- ✅ Special Offers promo banner with discount details
- ✅ About Us section with company stats
- ✅ Contact section with info cards + static contact form
- ✅ Footer with social links, quick links, and partner brands
- ✅ Hover animations and smooth transitions throughout
